# Yuteng.exe

A retro operating-system-inspired portfolio rebuilt with Next.js, React, and Tailwind CSS.

## Stack

- Next.js App Router
- React 19
- Tailwind CSS v4
- Static export for GitHub Pages deployment

## Pages

- `/` Home desktop
- `/portfolio` project archive
- `/blog` writing archive
- `/contact` contact console

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The static export is generated in `out/`.
